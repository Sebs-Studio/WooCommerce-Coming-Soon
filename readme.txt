== Description ==

Enables you to display a message to customers saying the product is "Coming Soon" rather than "Out of Stock".

== Installation ==
Installing "WooCommerce Coming Soon" can be done either by searching for "WooCommerce Coming Soon" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard.
3. Activate the plugin through the 'Plugins' menu in WordPress.
